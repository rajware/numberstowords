Namespace: numberstowords

numberstowords

This namespace exposes methods for converting numbers to words. For backward compatibility, it also exposes two properties, options and words, which can be set globally.

Members

(static) defaultOptions

Default options for numberstowords. Read-only.

(static) defaultWords

The default set of words used for unit conversition, plus words like 'only' and 'and'. Read-only.

(static, readonly) options

The global options for numberstowords. The option values can be changed, and subsequent calls to toWords, toIndianWords or toInternationalWords will use the changed values by default. This is maintained for backward compatibility, but deprecated. Pass an Opts object in the parameters instead.

Deprecated:
  • Yes
Examples
numberstowords.options.integerOnly = false;
numberstowords.options.decimalPlaces = 4;
numberstowords.options.useOnlyWord = true;

(static, readonly) words

The global set of words used for unit converstion, as well as words like 'and' and 'only'. The values can be changed, and subsequent calls to toWords, toIndianWords or toInternationalWords will use the changed values by default. This is maintained for backward compatibility, but deprecated. Pass a Words object in the parameters instead.

Deprecated:
  • Yes
Examples
numberstowords.tenWords[3] = 'treinta';
numberstowords.andWord = 'y';
